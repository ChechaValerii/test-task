## Requirements
- docker
- docker-compose
- node >= 16
- npm >= 8
- mongodb >= 4.0
- typescript >= 4.x
- redis >= 4.x

## Installation
```bash
npm install
```

## Run development server with docker-compose
```bash
docker-compose up -d
```

## Run development server without docker-compose
```bash
npm run start:dev
```

## Swagger Documentation
You can find the Swagger documentation at [http://localhost:3000/api/](http://localhost:3000/api/)
